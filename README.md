# 文件批量重命名工具 (File Renamer)

一个基于 Web 技术的现代化本地文件批量重命名工具，利用 **File System Access API** 直接操作本地文件，安全高效。

![Logo](./public/logo.svg)

## ✨ 主要特性

*   **🛡️ 安全隐私**：纯前端运行，无需上传文件，直接读写本地文件系统。
*   **📂 灵活模式**：
    *   **文件模式**：支持拖拽文件夹递归读取所有文件。
    *   **文件夹模式**：支持直接对文件夹本身进行批量重命名。
*   **⛓️ 强大的规则链**：
    *   **查找替换**：支持字符串和正则表达式 (Regex) 替换。
    *   **添加/插入**：在头部、尾部或指定位置插入文本。
    *   **删除**：按索引或长度删除指定字符。
    *   **序列号**：自定义起始值、增量、填充位数和位置。
    *   **格式化**：大小写转换、去除空格 (Trim)。
    *   **扩展名修改**：一键修改文件后缀。
*   **🎨 丝滑交互**：
    *   支持**拖拽排序**规则步骤（带平滑动画）。
    *   支持**拖拽文件/文件夹**直接添加。
    *   实时预览重命名效果，差异高亮显示。
    *   支持批量勾选/反选控制。

## 🛠️ 技术栈

*   **Vue 3**: 渐进式 JavaScript 框架
*   **Vite**: 下一代前端开发与构建工具
*   **Tailwind CSS**: 原子化 CSS 框架
*   **File System Access API**: 现代浏览器文件系统访问能力

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 构建生产版本

```bash
npm run build
```

## ⚠️ 兼容性说明

由于使用了 **File System Access API**，本工具仅支持基于 Chromium 的现代浏览器（如 Chrome, Edge, Opera 等）。Firefox 和 Safari 目前可能仅支持有限的功能或需要 Polyfill。
